# 🔧部署

1. 复制`.env.development.example` 为`.env.development`

2. 修改其中`TARO_APP_ID`, `TARO_APP_NAME`, `TARO_APP_API_HOST`的值成你自己的
   ![build.jpg](docs/images/build.png)

3. 执行`pnpm install` 安装依赖

4. 执行`pnpm build:weapp`进行构建

5. 用微信开发者工具打开dist目录
